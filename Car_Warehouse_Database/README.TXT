PROJECT TITLE: USED CAR WAREHOUSE DATABASE SYSTEM
PURPOSE OF PROJECT: To create a database for used cars so that users can use this to perform CRUD operations on the database system.
                    This database can perform search, add, delete and edit cars in the warehouse.
VERSION or DATE: v3, 27th May 2021
HOW TO START THIS PROJECT: Make an object of the car warehouse class and start with the processMainMenuResponse method.
AUTHORS: Kapish Kuchroo
USER INSTRUCTIONS: Make an object of carWarehouse and start with the only method which is processMainMenuResponse.
                   It will guide you through the database system and help you perform CRUD operations on the warehouse.
                   Please follow the instructions on the screen and put appropriate inputs.
